Curent Version: 1.0
Setup instructions
-------------------
1. Goto Awesome Core -> services module and add following lines, and save (most of these lines will be present, make sure last line is added if missing.)

[services.add echarts service_label='E Charts' post_type='echarts' desc='Echarts Service' /]

2. Now import e-charts.xml via wordpress importer.

In case of updating

1. Trash the echarts modules and empty the trash
2. Import this xml file.
